# Patch Registry

This folder stores active and historical SharpEdge patch notes.

## Postgame Agent usage

The SharpEdge Postgame Agent should read the latest patch note before every postmortem and classify the result as:

- keep
- modify
- revert

## Priority markets

Rebounds remain the highest-priority market. Patch notes should explicitly track rebound environment, shot profile, pace, lineup size, box-out matchups, foul trouble, and minute volatility.
